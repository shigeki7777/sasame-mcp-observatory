# Correct or remove a listing

All listings are built from **public-registry data**. We honor corrections and removals, no questions asked.

- **Remove:** email **consulting@srl-sasame.com** with the server URL. We delist it (and stop re-auditing it).
- **Correct:** tell us what's wrong (name, description, category, grade). If you believe a grade is wrong, the audit is reproducible — point us at the discrepancy and we'll re-run and fix the instrument in public if it's our error.
- Open an issue on this repo for anything public.

We never publish a "dead/worst/fake" list, and `unclaimed` always means *auto-indexed, unverified* — not a quality judgement.
